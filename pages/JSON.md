- # Introduction
  JSON (JavaScript Object Notation) is an open standard file format and data interchange format
- # Data Types Syntax
	- ## 1. Strings
	  ```json
	  {"name":"John"}
	  ```
	- ## 2. Numbers
	  ```json
	  {"age":30}
	  ```
	- ## 3. Objects
	  ```json
	  {
	  "employee":{"name":"John", "age":30, "city":"New York"}
	  }
	  ```
	- ## 4. Arrays
	  ```json
	  {
	  "employees":["John", "Anna", "Peter"]
	  }
	  ```
	- ## 5. Booleans
	  ```json
	  {"sale":true}
	  ```
	- ## 6. Null
	  ```json
	  {"middlename":null}
	  ```
