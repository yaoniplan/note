- #### Use "regex"
    - `\.(mp3|m4a|wav|wma|flac|aac|ogg)$` # Check if a file is an audio file
    - `\.[^\.]+$` # Extract file extensions
- ***Notes***
    - `\` # Escape
    - `[]` # Character class (Match any single character from the list inside)
    - `^` # Negation (Except)
    - `+` # Quantifier (One or more)
    - `()` # Group
    - `|` # OR (One of them)
    - `$` # At the end of the string
