- #### Rename file in batches
    - `for file in *.m4a; do mv "$file" "$(echo "$file" | sed -E 's/\[[0-9]+\]//g')"; done`
- ***Notes***
    - `-E` # Extended regular expressions
    - `[0-9]` # Any single digit from 0 to 9
    - `+` # One or more times
