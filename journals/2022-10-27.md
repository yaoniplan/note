- #### Add string to the end of specified lines in Vim
    - `:'<,'>s/$/<\/li>/`
- ***Notes***
    - `:'<,'>`
        - `V` # Select lines in visual line mode
        - `:` # See the format
    - `s` # Substitute
    - `$` # The end of line
    - `<\/li>` # Replace it with your desired string
        - `backslash` # Escape character
