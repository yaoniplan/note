- #### Use "jq"
    - `echo "$response" | jq -r '.content'`
        - `echo "$response" | jq .` # Get the key (e.g. `content`)
- ***Notes***
    - `jq` # JSON query
    - `-r` # Raw output
    - `nix profile install nixpkgs#jq` # Install dependencies
    - Because to extract the value of the `content` key from the JSON response.
