- #### Use "swappy"
    - `swappy --file ~/example.png` # Edit this file
    - `wl-paste | swappy --file -` # Edit from the clipboard
- ***Notes***
    - `Ctrl-s` # Save to the home directory
    - Mosaic (To shield private data)
        - `r` # Rectangle
        - `f` # Fill shape
        - `Ctrl-c` # Copy to the clipboard
        - `q` # Quit (The same as `Esc`)
