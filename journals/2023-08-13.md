- #### Output chess board in C
    - `vi ~/chessBoard.c`
      ```
      #include <stdio.h>
      
      int main() {
          for (int i = 0; i < 8; i++) {
              for (int j = 0; j < 8; j++) {
                  if ((i + j) % 2 == 0)
                      printf("██"); // Output white square
                  else
                      printf("  "); // Output black square with two spaces
              }
              printf("\n");
          }
          return 0;
      }
      ```
