- Push using [[Git]] automatically
	- "~/.config/shell/autoGit.sh"
	  ```sh
	  #!/bin/sh
	  while true; do
	    cd ~/note
	    sleep 20
	    git add *
	    sleep 20
	    git commit -m "Update at `date +%F-%T`"
	    sleep 20
	    git push
	  done
	  ```
	- "~/.config/i3/config"
	  ```
	  # Git automatically
	  exec ~/.config/shell/autoGit.sh
	  ```
- ***Notes***
	- `chmod u+x ~/.config/shell/autoGit.sh` # Add executable permission
