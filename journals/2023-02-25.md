- #### Search for selected text in Vim
    - `/` # Search mode
    - `C-r` # Register
    - `"` # The default register
- ***Notes***
    - `y` # Yank text to the `"` register before entering search mode
        - `yl`
    - Because to search languages in other countries (e.g. *Japanese*).
