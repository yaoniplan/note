- #### Use "streamlink"
    - `streamlink --stream-segment-threads 3 --player-no-close --player mpv --player-args '--speed=2' "https://example.com/index.m3u8" best`
- ***Notes***
    - `--stream-segment-threads 3` # Use 3 threads to download segments in parallel
    - `--player-no-close` # Prevent Streamlink from terminating the player when the stream ends
    - `--player-args '--speed=2'` # Pass arguments to the player
    - Because to speed up buffering for an HLS link. (Make playback smoother)
