- #### Use "hyprctl" 
    - `hyprctl -j activewindow | jq -r '"\(.at[0]),\(.at[1]) \(.size[0])x\(.size[1])"' | `
        - e.g. `690,22 654x355`
- ***Notes***
    - `-j` # JSON
    - `\()` # Placeholders (Be replaced with actual values)
    - `.at[0]` # The first element (e.g. `690`) of the array (e.g. `at`)
