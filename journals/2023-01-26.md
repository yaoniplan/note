- #### Remap keys in Vim
    - `vim ~/.vimrc`
      ```
      nnoremap <C-H> <C-W>h
      nnoremap <C-J> <C-W>j
      nnoremap <C-K> <C-W>k
      nnoremap <C-L> <C-W>l
      ```
- ***Notes***
    - `C-H` # Use `Ctrl-h` to move cursor to the left panel
    - `<C-W>h` # Use `<Ctrl-w>h` to move cursor to the left panel too
- ***References***
    - https://superuser.com/questions/280500/how-does-one-switch-between-windows-on-vim
