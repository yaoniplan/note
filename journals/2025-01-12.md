- #### Use "yazi"
    - `yazi`
        - `S` # Search file contents in the current directory and subdirectories (Use `rg`)
        - `.` # Show or hide dotfiles
        - `q` # Quit
        - `~` # Open the help menu
        - `g<Space>` # Go to the path you want to go
        - `cc` # Copy file path (The same as `realpath`)
- ***Notes***
    - `ya pack --add grappas/wl-clipboard` # Install this plugin if clipboard doesn't work
    - `vim ~/.config/yazi/keymap.toml` # Configure keybindings
      ```
      [[manager.prepend_keymap]]
      on   = [ "g", "b" ]
      run  = "cd ~/.local/bin"
      desc = "Goto ~/.local/bin"

      [[manager.prepend_keymap]]
      on   = [ "<C-j>" ]
      run  = "open"
      desc = "Open file or enter directory"
      ```
    - `v` # Visial mode (Selection mode, select multiple and delete in batches)
    - Delete a directory
        - `/tachi` # Search for the directory you watn to delete
        - `d` # Delete (Move it to `~/.local/share/Trash/`)
        - `y` # Yes
    - Because to use the fastest file manager.
    - `nix profile install nixpkgs#yazi` # Install dependencies
