- #### Fetch an icon of a website
    - `https://favicone.com/google.com?s=32`
- ***Notes***
    - `google.com` # Replace it with other domain name of a website
    - `s` # Size
    - `32` # 32x32 pixels (Clearer)
    - `https://api.faviconkit.com/google.com/64` # Another method
