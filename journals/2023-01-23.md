- #### Replace context of the current line in Vim
    - `:s/italic/boldAndItalic/g`
- ***Notes***
    - `s` # Substitute
    - `g` # Global
    - `Shift-v` # Select some line
        - `:` # Press it
        - `s/italic/boldAndItalic/g` # Type it 
- ***References***
    - `:help :substitute`
